## 建議的 coding 格式
#### `pylint`
<details><summary markdown="span">使用 `pylint` 檢查 程式碼</summary>
* 透過 `pip install pylint` 安裝<br/>
* `pylint XXX.py` 執行<br/>
* `pylint` 並非完美, 僅是一個輔助工具. 你應該事情況<br/>
** 修改程式碼<br/>
** 將部分報錯 加入 ignore list (過多報錯, 可能導致你忽略真正需要修改的資訊)<br/>
</details>
